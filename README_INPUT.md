## Guía de uso para el archivo "input.txt"

El archivo "input.txt" nos permite seleccionar los parámetros de la interacción que se calculará en el código principal. Está compuesto de cinco líneas, tal que cada una corresponde a un parámetro diferente:

Línea 1: elige el tipo de interacción entre 0 (electromagnética), 1 (corrientes débiles cargadas) y 2 (corrientes débiles neutras)

Línea 2: elige el blanco entre 0 (neutrón) y 1 (protón)

Línea 3: determina la energía de la partícula disparada en MeV

Línea 4: indica la helicidad entre 1 (antineutrino) y -1 (neutrino)

Línea 5: elige la parametrización de los factores de forma entre 1 (Galster), 2 (Galster para Q^2<<), 3 (Kelly), 4 (Arrington-Sick), 5 (GK) y 6 (GKex)
